Introduction

This document provides an overview of front-end web basics — HTML, CSS, and JavaScript — and best practices.

It is meant to be a compact technical reference for beginners and for quick revision.

You can navigate via the sidebar links to jump to a specific section.

<!doctype html> <meta charset="utf-8">
HTML Basics

HTML (HyperText Markup Language) structures the content of web pages.

Common tags include headings, paragraphs, lists, links, images, and forms.

Example of a paragraph and an image tag:

<p>Your paragraph text</p> <img src="photo.jpg" alt="description">
  • Headings: <h1> ... <h6>
  • Paragraphs: <p>
  • Links: <a href="...">
CSS Basics

CSS (Cascading Style Sheets) describes how HTML elements are displayed.

Use selectors to target elements and properties to style them (color, layout, spacing).

body { font-family: Arial, sans-serif; } .container { display: grid; }

CSS also supports media queries to build responsive layouts.

  • Box model: margin, border, padding, content
  • Display: block, inline, inline-block, flex, grid
JavaScript Overview

JavaScript brings interactivity to the web — DOM manipulation, event handling, async operations.

Keep logic separate (scripts) and avoid deeply coupling behavior and presentation.

document.querySelector('#id') fetch('/api/data').then(res => res.json())

Good practices: modular code, use let/const, avoid global pollution.

  • Event listeners
  • Promises / async-await
  • Modules (ES6 import/export)
Accessibility and SEO

Accessibility ensures everyone can use your site (alt attributes, semantic tags, keyboard navigation).

SEO helps search engines understand your content: headings, meta tags, structured data.

<img alt="..."> <nav role="navigation">

Use semantic HTML and ARIA attributes where appropriate. Test with keyboard-only and screen reader tools.

  • Use semantic tags: header, nav, main, footer
  • Provide meaningful alt text for images
  • Ensure good color contrast

That's a quick overview. Keep learning and iterate on accessibility as you build.
